# Instructions

1. Open the "Game.sln" VS Community solution found in "Game"
2. Run the game and play for a bit (recommended: skip the tutorial and play at least 2 levels)
3. Move the telemetry file stored in "Game\Game\src\data\telemetry" to "analytics\telemetry_data"
4. Open and run "analytics/analytics.py"
	- The exact numbers will be displayed through the console
	- Various graphics will popup with each analized section
